# TBH 素材・装備メモ

- `index.html`: 画像付きメモ本体
- `keep-materials.html`: 残す素材一覧
- `tools/build_site.py`: 抽出済みデータから `index.html` を再生成するスクリプト
- `tools/update_market.py`: 市場から日本円の最低価格・中央値・取引量を取得するスクリプト
- `.github/workflows/update-market.yml`: 毎日05:15(JST)ごろに価格を更新して公開ページを再生成
- `guide-effects.json`: コミュニティガイドから整理した装飾/彫刻の数値、ソケット、特殊装備効果
- `icons/`: ゲーム本体から抽出したアイコン
- `images/`: 市場から取得した掲載アイコン
- `characters/`: キャラクター別ページ
- `ItemTable.csv` / `StringTable.csv`: 抽出元のローカライズ表
- `market-items.json`: 市場検索結果と円建て価格

## 入っているもの

- 511件の素材/装備カード
- アイテムクリック時の用途・売却判断・関連レベル帯・マーケット表示
- 一覧枠だけを独立スクロールする素材/装備ビュー
- 分類、ビルド、等級、売却判断で絞り込むフィルター
- 素材とレベル帯の整理表
- 残す素材一覧ページ
- キャラクター別ページ
- キャラクター別の装備/残すステータスメモ
- キャラクター別の残す付与素材表
- クラフト/売却判断表
- 物理/攻撃速度、魔法/属性、防御/耐久のビルド別メモ
- 装飾/彫刻素材ごとの武器・防具・アクセ向け数値表
- レベル/必要等級つきの特殊装備効果一覧
- 市場の日本円価格、中央値、取引量

公開データで安定取得できる価格は最低価格・中央値・取引量です。現在の最高売注文は未取得として表示しています。
価格は変動するため、売却前に市場で再確認してください。
装飾/彫刻の数値と特殊装備効果はコミュニティガイドを補助情報として使い、名称はローカルのItemTable/StringTableを優先しています。
